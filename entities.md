# Entities

### User

It represents a system user.

| Field        | Type     |
| ------------ | -------- |
| Id           | Guid     |
| Name         | string   |
| Surname      | string   |
| Email        | string   |
| Password     | string   |
| IsArtist     | boolean  |
| IsActive     | boolean  |
| DateRegister | datetime |
| Role         | string   |
